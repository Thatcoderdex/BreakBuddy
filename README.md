# Break Buddy

Chrome and Microsoft Edge extension for break reminders with:

- interval reminders
- in-page fullscreen overlays
- quiet hours
- soft / normal / aggressive modes
- task-specific sounds
- focus lock mode
- snooze
- streaks and break history

## Load in Edge or Chrome

1. Open `edge://extensions` or `chrome://extensions`
2. Turn on `Developer mode`
3. Click `Load unpacked`
4. Select this folder

## Main files

- `manifest.json`
- `background.js`
- `popup.html`
- `popup.js`
- `options.html`
- `options.js`
- `content.js`
- `content.css`
- `ui.css`

## Current features

- quick popup controls
- custom reminder prompts
- full-screen stand-up mode
- aggressive reminder motion and sound
- countdown ring
- per-style countdown timing
- break completion stats
- streaks
- recent history
- hydration / movement / screen-rest balance bars
